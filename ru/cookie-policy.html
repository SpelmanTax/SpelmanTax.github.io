Политика использования файлов cookie

1. Что такое файлы cookie

Файлы cookie — это небольшие текстовые файлы, которые используются сайтами для обеспечения базовых функций и корректной работы.

2. Cookie, используемые на данном сайте

Технические cookie: необходимы для работы сайта.
Не используются: cookie для профилирования, маркетинга, аналитики, а также cookie сторонних сервисов.

3. Cookie GitHub Pages

GitHub Pages может использовать технические cookie в целях безопасности и функционирования сайта. Эти cookie не отслеживают поведение пользователей.

4. Управление файлами cookie

Пользователь может управлять cookie через настройки своего браузера. Так как на сайте отсутствуют не‑технические cookie, баннер согласия не требуется.
